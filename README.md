# Atlas

**Atlas** is a modular engineering workbench for Linux designed to visualize, analyze, and simulate telemetry from embedded and hardware systems.

Built with modern C++, Atlas provides a unified environment for importing data, exploring time-series signals, running domain-specific analyses, and extending functionality through a flexible plugin architecture. Rather than creating separate tools for every project, Atlas serves as a single platform that adapts to different engineering workflows.

Whether you're analyzing a model rocket flight, tuning a 3D printer, debugging a robotics project, or inspecting data from custom embedded hardware, Atlas aims to provide the tools needed to understand complex systems through intuitive visualization and powerful analysis.

### Core Features

* Interactive telemetry visualization with synchronized graphs and tables
* Modular plugin system for domain-specific analysis
* Workspace management for organizing engineering projects
* Support for multiple data formats and live telemetry streams
* Simulation and reconstruction tools for supported plugins
* Extensible architecture designed for custom workflows

### Vision

Engineering projects often rely on a collection of disconnected tools for logging, plotting, simulation, and analysis. Atlas brings these capabilities together into a single extensible platform, allowing engineers, makers, and students to spend less time switching software and more time understanding their systems.

Atlas is built from the ground up with performance, extensibility, and usability in mind, making it suitable for everything from personal experiments to large engineering projects.
